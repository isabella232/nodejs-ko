Node v6.10.3(LTS)

  • 작성자: Myles Borins
  • 원문: Node v6.10.3 (LTS)
  • 번역: taggon

주요 변경사항

  • module
    • 윈도우즈에서 모듈 로딩 전역 폴백이 올바르게 Node 실행 파일이 있는 디렉터리를 가리키도록 수정했습니다. (Richard Lau) #9283
  • src
    • 굉장히 드문 사례에서 발생하는 base64 디코딩 문제를 수정했습니다. (Nikolai Vavilov) #11995
  • tls
    • TLS를 사용할 때 드물게 발생하는 세그멘테이션 결함(segmentation faults)을 수정했습니다.
      • (Trevor Norris) #11947
      • (Ben Noordhuis) #11898
      • (jBarz) #11776
